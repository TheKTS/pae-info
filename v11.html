업데이트 개요

PAE Core Version 11은 기존 v10의 레거시 구조를 전면 재설계한 대규모 업데이트입니다. 비구조화된 코드를 도메인별로 분류하고, 클라우드 DB를 도입하여 시스템 관리·배포·권한·로깅을 자동화했습니다.

이번 업데이트를 통해 PAE는 단순한 자동화 도구에서 체계적으로 관리되는 엔터프라이즈 플랫폼으로 진화했습니다.

모듈 아키텍처 재설계

비구조화된 레거시 코드(CM_/CF_ 접두어)를 도메인별로 분류하고, 54개 모듈을 계층적으로 재구성. 유지보수성과 확장성을 대폭 향상.

54개 모듈 · 도메인별 구조화
☁

클라우드 DB 도입

Supabase 기반 8개 테이블을 신설하여 시스템 메타정보, 버전, 권한, 에러 로그, 배포 이력 등을 클라우드에서 통합 관리.

Supabase · 8개 테이블
🚀

배포 자동화

7단계 배포 파이프라인을 구축하여, 개발 완료 후 원클릭으로 시스템 배포 · 버전 등록 · 이력 기록까지 자동 수행.

7단계 원클릭 배포
🔒

사용자 권한 체계

기존 하드코딩 방식을 DB 기반 권한 테이블로 전환. 시스템별 접근 제어, 개인정보 동의 이력까지 체계적으로 관리.

DB 기반 권한 CRUD
📋

시스템 관제 강화

에러 로그 자동 기록, 로그인 이력 추적, 4개 카테고리 시스템 진단 기능을 도입하여 실시간 운영 모니터링 체계 구축.

에러 로그 · 진단 · 모니터링
🌐

API 연계 중앙화

Dooray · W-ERP · SNC-ERP 등 외부 시스템 연계를 중앙화된 Core/Endpoint 패턴으로 통합. 인증 자동 주입, URL 빌더 표준화.
