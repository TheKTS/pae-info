PAE

Process Automation by Excel
PAE란?

PAE(Process Automation by Excel)는 복잡하고 반복되는 업무를 체계적으로 정비하고, 시스템 자동화를 통해 업무 효율을 극대화하는 전사 업무 자동화 플랫폼입니다.

우리관리 업무개선팀이 기획 · 개발한 Excel 기반 프로세스 자동화가 전사 업무 혁신을 선도합니다.

📊

Excel 기반

별도 설치 없이 익숙한 Excel 환경에서 바로 사용

🔗

시스템 연계

W-ERP, Dooray, 문자나라 등 다양한 외부 시스템과 연동

⚡

전사 적용

사무행정직 대부분의 업무를 자동화 가능

2022. 03

PAE 시스템 첫 도입

자동계산 및 보고서 양식 생성 기능으로 전사 업무 자동화의 첫 걸음을 내딛다.

전사 업무 자동화 시작
2023. 01

로그인 방식 변경

WINE ID 로그인 및 W-ERP 사용자 인증 TOKEN 체계를 도입하여 보안과 접근성을 강화.

로그인 및 인증 체계 개선
2023. 02

W-ERP API 개설 및 활용

ERP 시스템과 데이터를 직접 연계하여 수작업 데이터 입력을 제거.

ERP 연계 데이터 자동화
2023. 06

Dooray API 활용

시스템 버전별 원격 관리, 다중 시스템 연계, 알림 기능을 제공하여 업무 관리를 자동화.

업무 관리 자동화 · 원격 관리
2023. 07

문자나라 시스템 연계

Excel에서 추출한 데이터를 기반으로 문자 발송을 자동화.

외부 커뮤니케이션 자동화
2024. 10

BPO팀 창설 (PAE 기반 운영 설계)

PAE 시스템을 핵심 인프라로 BPO팀을 신설. 팀 운영의 80% 이상을 PAE로 구동하며, 인당 관리 사업장을 40개 → 90개로 목표 설정.

PAE 기반 조직 설계 · 인당 생산성 2배 이상 목표
2024 하반기 🏆 성과

월간 업무절감 12일분 돌파

PAE 시스템 20개 가동, 월간 업무절감 효과가 약 12일분(8시간/일 기준)을 돌파.

연간 약 150일분 업무 절감 달성
2025. 07

신규 직원 증원

업무개선팀 개발 인력을 확충하여 개발 · 운영 역량을 강화.

개발/운영 지원 강화
2025. 11

System Revamp (Major Version 10)

버전 관리 체계를 전면 재설계하고 협업 환경을 강화하는 대규모 시스템 개선을 단행.

협업 최적화 · 시스템 전면 개선
2026. 01 🏆 성과

BPO팀 인당 90개 사업장 달성

2024년 10월 창설 시 설정한 인당 관리 사업장 90개 목표를 15개월 만에 달성. 기존 40개 대비 2.25배 생산성 향상.

40 → 90개/인
PAE 기반 생산성 혁신 입증
2026. 01 🏆 성과

월간 업무절감 18일분 돌파

PAE 시스템 25개 가동 기준, 월간 업무절감 효과가 약 18일분(8시간/일 기준)을 돌파. 연간 기준 약 218일분으로 실질적 인력 1명분에 해당.

218일/년
연간 인력 1명분 업무 절감 효과
2026. 01

업무개선팀 신설

경영개선팀에서 업무개선팀으로 개편. 팀장 1명 + 신규 2명 체제로 전환하여 개발·운영 역량 확대.

조직 개편 · 전담 인력 확보
2026. 03

AI 기반 개발 체계 도입

AI가 코드·문서·테스트 초안을 생성하고, 팀원이 검증하는 2단계 워크플로우를 확립. IT 경험이 없는 신규 인력도 즉시 투입 가능한 구조를 구축.

개발 생산성 약 3배 향상 · 탈속인화
PAE Core Version 11 — 주요 변화
v10 → v11 Major Update

2025년 11월, PAE 핵심 엔진을 전면 재설계하여 체계적이고 확장 가능한 구조로 전환했습니다.

🏗

모듈 아키텍처 재설계

비구조화된 레거시 코드를 도메인별로 분류하고, 54개 모듈을 계층적으로 재구성. 유지보수성과 확장성을 대폭 향상.

54개 모듈 · 도메인별 구조화
☁

클라우드 DB 도입

Supabase 기반 8개 테이블을 신설하여 시스템 메타정보, 버전, 권한, 에러 로그, 배포 이력 등을 클라우드에서 통합 관리.

Supabase · 8개 테이블
🚀

배포 자동화

7단계 배포 파이프라인을 구축하여, 개발 완료 후 원클릭으로 시스템 배포·버전 등록·이력 기록까지 자동 수행.

7단계 자동 배포
🔒

사용자 권한 체계

기존 하드코딩 방식을 DB 기반 권한 테이블로 전환. 시스템별 접근 제어, 개인정보 동의 이력까지 체계적으로 관리.

DB 기반 권한 관리
📋

시스템 관제 강화

에러 로그 자동 기록, 로그인 이력 추적, 4개 카테고리 시스템 진단 기능을 도입하여 실시간 운영 모니터링 체계 구축.

에러 로그 · 진단 · 모니터링
🌐

API 연계 중앙화

Dooray·W-ERP·SNC-ERP 등 외부 시스템 연계를 중앙화된 Core/Endpoint 패턴으로 통합. 인증 자동 주입, URL 빌더 표준화.
