PAE Core Version 10

구조 소개
PAE-10 · Legacy Core · 2022 ~ 2025
개요

PAE Core Version 10은 2022년 PAE 시스템 첫 도입부터 2025년 11월 v11 전환까지 약 3년간 운영된 코어 엔진입니다.

기능별로 세분화된 모듈 구조를 채택하여, 초기화(Init)와 실행(Execute)을 분리하는 패턴으로 설계되었습니다. 총 93개 모듈이 평면 구조로 배치되어 있으며, 25개 PAE 시스템의 기반이 되었습니다.

Init / Execute 분리 패턴

폼과 주요 기능을 초기화(_Init)와 실행(_Execute)으로 이원화. 시스템 관리, 로그인, 컨트롤 관리 등에 적용.

SysMng_Init + SysMng_Execute
🛠

Dev / Prod 이원 구조

로그인 로직에서 개발 모드와 프로덕션 모드를 완전히 분리. 각각 별도의 Init/Execute 모듈로 관리.

Login_Init_Dev · Login_Init_Prod
🎯

기능별 세분화

하나의 기능을 여러 모듈로 세분화하여 관리. 컨트롤 관리만 5개, 시스템 관리 4개, API별 최대 8개 모듈로 구성.

세밀한 관심사 분리
